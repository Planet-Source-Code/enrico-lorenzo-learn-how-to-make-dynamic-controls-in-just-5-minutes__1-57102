Learn How To Make Dynamic Controls In Just 5 Minutes

 

What is a dynamic control?

A dynamic control is a control created during runtime.

 

How do I create a dynamic control?

Specs: Create a CommandButton control with a caption “Click Me” everytime the form loads. The control should have an event that when clicked, a message box will show “This is a dynamic control”.

 

Option Explicit

 

Private WithEvents cmdDynamic As CommandButton

 

Private Sub Form_Load()

Set cmdDynamic = Controls.Add("VB.CommandButton", "Command1")

 

cmdDynamic.Caption = "Click Me"

cmdDynamic.Visible = True

End Sub

 

Private Sub cmdDynamic_Click()

MsgBox "This is a dynamic control."

End Sub

 

Option Explicit

Used at module level to force explicit declaration of all variables in that module.

 

Private

Used at module level to declare private variables and allocate storage space.

 

WithEvents

Keyword that specifies that varname is an object variable used to respond to events triggered by an ActiveX object.

Without WithEvents, the event cmdDynamic_Click() will not work.

 

Set

Assigns an object reference to a variable or property.

 

“VB.CommandButton” - ProgID

A text string with no spaces that identifies an object class.
